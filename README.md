# 🎲 برا السالفة — النسخة اللبنانية (Bara Al-Selfe)

![Logo](assets/images/bara-al-selfe-logo.png)

**Bara Al-Selfe — Lebanese Edition** is a fast-paced, social deduction game designed for friends and families. It's a "pass-and-play" experience where everyone knows a secret word except for one person—who is **"Outside the Story" (برا السالفة)**.

---

## 🇱🇧 Overview / لمحة عن اللعبة
لعبة "برا السالفة" بنسختها اللبنانية هي لعبة ذكاء، تمويه، وتسلية. الكل بيعرفوا "الكلمة السرية" إلا واحد... هيدا هو اللي "برا السالفة". هدفه يكتشف شو هي الكلمة من دون ما ينفضح، وهدف البقية يكشفوه!

---

## ✨ Features / المميزات
- **Lebanese Dialect:** Fully localized content, categories, and humor for the Lebanese community.
- **Pass & Play:** No internet required! Play with 3 to 10+ players on a single device.
- **Dynamic Categories:** Choose from diverse topics like Lebanese food, animals, landmarks, and more.
- **Modern UI/UX:** A stunning, premium design with smooth animations, glassmorphism, and vibrant Lebanese tones.
- **Offline First:** Perfect for gatherings, cafes, and road trips.

---

## 🎮 How to Play / كيف بتلعب؟
1. **إختار الفئة:** من الأكل اللبناني للحيوانات أو الأماكن المشهورة.
2. **وزع الأدوار:** كل حدا بياخد التليفون وبيشوف "الكلمة السرية" إلا واحد رح تطلعله عبارة "إنت برا السالفة".
3. **بلش الأسئلة:** كل حدا بيسأل التاني سؤال صغير عن الكلمة. لازم تكون ذكي! إذا كنت "جوا"، ما تحكي زيادة لتفضح الكلمة. وإذا كنت "برا"، حاول تموّه لتبيّن إنك بتعرف.
4. **كشف المستور:** بعد جولة الأسئلة، الكل بيصوت على الشخص اللي شاكين فيه إنه "برا السالفة".

## 📦 Game Categories / الفئات
The game features thousands of words across these specific Lebanese categories:
- 🥙 **أكل لبناني (Lebanese Food):** From Hummus and Tabbouleh to Mansaf and Kibbeh.
- 🐾 **حيوانات (Animals):** Local and global animals in Lebanese terminology.
- 🥦 **خضار وفواكه (Vegetables & Fruits):** Fresh produce from our local markets.
- 👗 **ملابس (Clothes):** Traditional and modern Lebanese fashion.
- 🍮 **حلويات (Desserts):** Knefe, Baklawa, and all the sweet treats.

## 🛠️ Getting Started / تشغيل المشروع

### Prerequisites
- Node.js (Latest LTS)
- Expo Go app on your phone (or an emulator)

### Installation
1. **Clone the repository:**
   ```bash
   git clone https://github.com/MahmoudAmouni/bara-al-selfe-LebaneseEdition.git
   ```
2. **Install dependencies:**
   ```bash
   cd baraAlSelfe-LebansesEdition
   npm install
   ```
3. **Start the development server:**
   ```bash
   npx expo start
   ```
4. **Play!** Scan the QR code with your phone and enjoy.
